📘 Study Help

Useful notes and tips for different school subjects:

Mathematics

  • Area of Circle = πr²
  • Pythagoras Theorem: a² + b² = c²
  • Quadratic Formula: x = (-b ± √(b² - 4ac)) / 2a

English

  • “Their” = belongs to them, “There” = a place, “They’re” = they are
  • Good essays: Introduction → Body → Conclusion
  • Read daily to improve vocabulary

Science

  • Water = H₂O
  • Newton’s First Law: An object in motion stays in motion unless acted on by force
  • Photosynthesis: Plants make food using sunlight, CO₂, and water

Geography

  • The Nile is the longest river in the world
  • Mount Everest is the tallest mountain
  • Deserts cover about 33% of Earth’s land

History

  • World War I: 1914–1918
  • World War II: 1939–1945
  • Egyptians built pyramids over 4,000 years ago

Computer

  • HTML = structure, CSS = style, JS = behavior
  • CPU = Central Processing Unit (the brain of computer)
  • Binary code uses only 0 and 1
